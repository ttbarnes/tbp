# Tony Barnes Portfolio
Built with AngularJS

##Prerequisites
- Ruby, compass
- node.js
- Yeoman (grunt, bower)

##Installation
1) Fork a copy of the app to your machine
2) Install the dependancies:
`bower install`
`npm install`

##Running the app
For example...

`grunt serve`

Run unit tests:
`grunt karma:unit`

Run e2e tests:
`grunt test`
