## Â Prerequisite

- Node version 18 or higher with corresponding npm.
- Run `npm install` or `yarn install` in the root directory.

## Tech stack

- Node, NPM
- Next.js, React
- [Jest](https://jestjs.io/)
- [ESlint](https://eslint.org)
- [framer-motion](https://www.framer.com/motion/)

## Running locally

```bash
npm run dev
# or
yarn dev
```

## Testing

### Execute unit tests

```shell
npm run test
```

### Watch the unit tests

```shell
npm run test:watch
```

## Linting

### Execute linting

```shell
npm run lint
```

### Fix linting errors

```shell
npm run lint:fix
```
